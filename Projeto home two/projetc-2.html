Belimon

Projeto de Identidade Visual

O desafio do projeto era criar uma identidade que previsse o momento da marca quando se tornar uma franquia. A etapa de pesquisa foi fundamental para entender os futuros pontos de contato e quais sistemas visuais fariam parte para a gestão dos franqueados.



Então a construção de uma personalidade foi essencial para conectar com os dois públicos, tanto empreendedoras quanto clientes finais.

• Diagnóstico de Marca • Estratégia de Marca • Identidade Visual • Plano de Ação • Mídias Sociais
